## Layout Strategy

| Option | Description | Selected |
|--------|-------------|----------|
| Fixed-width grid with horizontal scroll + fixed right panel | Grid has min-width, scrolls horizontally, panel fixed on right | |
| Responsive grid with collapsible/minimizable cards | Cards can be collapsed to title bar, grid auto-reflows | Initially selected, then revised |
| Full-height grid left, resolution panel below | Grid takes full viewport height left, panel below | |
| Narrow/compact triplet chips to fit all 4 columns | Each triplet as single-line chip, all 4 columns visible at 1280px+ | ✓ |

**User's choice:** Narrow/compact triplet chips to fit all 4 columns (after initially picking collapsible cards, then reconsidering)

**Notes:** User initially wanted responsive collapsible cards but then said "I'm sorry I don't want collapsable triplet cards. Can we go back?" — revised to compact chip approach. No collapsible/minimizable cards. All 4 columns always visible.
