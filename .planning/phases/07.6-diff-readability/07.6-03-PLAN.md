<test_workaround>
### React 19 Test Workaround (MANDATORY)

All React component tests in this project MUST use the following pattern — NOT `@testing-library/react`:

```typescript
import ReactDOMClient from 'react-dom/client';
import ReactDOM from 'react-dom';

function render(el: React.ReactElement): { container: HTMLElement; root: ReactDOMClient.Root } {
  const container = document.createElement('div');
  document.body.appendChild(container);
  const root = ReactDOMClient.createRoot(container);
  ReactDOM.flushSync(() => { root.render(el); });
  return { container, root };
}
```

**Reason:** React 19.2.7 CJS bundle has `React.act` undefined, which crashes `@testing-library/react`'s `render()`. The workaround uses `createRoot` + `flushSync` directly.

**Test files with JSX must use `.tsx` extension** — vitest compiles `.tsx` with the React JSX transform.

The existing `ResolutionPanel.test.tsx` already follows this pattern. Do NOT break it.
</test_workaround>
